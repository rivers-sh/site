why you should use a window manager

why it can be more efficient than a desktop environment

* before beginning the yap session, i'm going to assume most people reading this know basic linux fundamentals, and kinda know how computers work

so, many people begin their linux journey with common distros (mint, ubuntu, etc.), these distros come with easy-to-understand desktop environments. eventually, however, you might find yourself yearning for more efficiency. this is when you might find yourself researching window managers.

some quick reasons why i prefer a window manager over a desktop environment

  • debloats my system (makes my old thinkpad run faster)
  • quicker to use (i can move, select, and resize windows with keyboard only)
  • more personal (i customize everything as i go)
  • overall better for single-monitor devices

so, which window manager is best for noobs? to be honest, there are way too many for me to individually go over each one, the main 3 that i recommend are:

  • i3wm (user friendly, good documentation, easy to refresh)
  • openbox (simple config file, ObConf is cool)
  • xfwm (very minimal learning curve, straightforward)

personally, though, i started with dwm, then jumped to i3wm because i found it easier, and now i'm on bspwm
